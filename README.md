# Road Damage Detection Using Deep Learning

## Overview

This project develops a deep learning model for automatically detecting road infrastructure issues from street images. Automated road damage detection can help municipalities monitor road conditions more efficiently while reducing manual inspection costs.

The model classifies road images into multiple categories using an EfficientNet-based convolutional neural network.

---

## Features

* Image preprocessing
* Multi-class road damage classification
* EfficientNet deep learning model
* Model evaluation
* Confusion matrix visualization
* Prediction examples

---

## Technologies Used

* Python
* TensorFlow
* Keras
* OpenCV
* NumPy
* Matplotlib
* Scikit-learn

---

## Dataset Classes

The model classifies images into:

* Broken Road Sign
* Damaged Road
* Illegal Parking
* Mixed Issues
* Potholes

---

## Workflow

1. Load road images
2. Image preprocessing and resizing
3. Train EfficientNet model
4. Evaluate performance
5. Predict road issue category

---

## Results

The model successfully learns visual features associated with different road defects and demonstrates the potential of deep learning for intelligent road maintenance systems.
